SHA-256 Hashing

This project is not designed to be a proper usage of the SHA-256 hashing algorithm, and is instead simply an example, and a learning experience for myself. This form will only store the hashed and salted password in local storage. It does not log you into anything, and, apart from the hashing and salting, is completly unsecure.
